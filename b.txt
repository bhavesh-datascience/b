Practical 1: Pivot Tables and Pivot Charts in Excel

Aim: Import data warehouse data into Microsoft Excel to create Pivot Tables and Pivot Charts.

Steps:

Download Data: Get the "AdventureWorks Sales.xlsx" file from the Microsoft Power BI desktop samples GitHub repository.
Initialize Pivot: Open the file, go to the Insert Tab > Pivot Table > From Table/Range and select New Workbook.
Build Table: Drag desired fields (e.g., ProductID, ProductName) from the field list into the "Filters", "Columns", "Rows", or "Values" sections .
Create Chart: Go to Insert Tab > Pivot Chart, select your preferred graph structure, and click OK.
 

 

Practical 2: What-If Analysis for Data Visualization

Aim: Apply What-if Analysis (Scenario Manager, Goal Seek, and Data Tables) to visualize different data outcomes.

Steps:

Data Entry: Create a table for "Book Store" data including "Total Books", "Highest Price", "Lowest Price", and "Unit Profit".
Access Tool: Navigate to the Data tab > What-if Analysis > Scenario Manager.
Define Scenarios: Click Add, name the scenario (e.g., "60%"), and change the relative percentage values (e.g., 0.6 for 60%, 0.7 for 70%).
Compare: Add scenarios up to 90% and click Show to see how results shift based on variables.

Operational Code:

R

# Using the built-in Iris dataset

newiris <- iris

newiris$Species <- NULL

kc <- kmeans(newiris, 3)

table(iris$Species, kc$cluster)

plot(newiris[c("Sepal.Length", "Sepal.Width")], col=kc$cluster)

points(kc$centers[,c("Sepal.Length", "Sepal.Width")], col=1:3, pch=8, cex=2)

 

Practical 5: Linear Regression (R Language)

Aim: Predict a continuous dependent variable based on independent variables.

Operational Code:

R:

x <- c(151, 174, 138, 186, 128, 136, 179, 163, 152, 131)

y <- c(63, 81, 56, 91, 47, 57, 76, 62, 48, 74)

relation <- lm(y ~ x)

print(summary(relation))

a <- data.frame(x = 170)

result <- predict(relation, a)

print(result)

png(file="linearregression.png")

plot(y, x, col="blue", main="Height & Weight", abline(lm(x ~ y)), cex=1.3, pch=16, xlab="Weight", ylab="Height in cm")

Practical 8: Advanced Data Visualization

Aim: Represent data pictorially using Python (Matplotlib/Seaborn) and Power BI.

8(a) Python Visualization

Python

import pandas as pd

import matplotlib.pyplot as plt

file_path = "AdventureWorks Sales.csv"

sales = pd.read_csv(file_path) 

print(sales.info())

print(sales.head())

 

plt.figure(figsize=(10,6))

sales_count = sales.groupby('Channel').count()['SalesOrderLineKey'] sales_count.plot(kind='bar', color=['skyblue', 'orange', 'green'])

plt.title('Total Sales Orders by Channel')

plt.xlabel('Channel')

plt.ylabel('Number of Orders')

plt.xticks(rotation=45)

plt.grid(axis='y', linestyle='--', alpha=0.7)

plt.show()


pie:

plt.figure(figsize=(8,8))

sales_count.plot(kind='pie', autopct='%1.1f%%', colors=['lightblue', 'lightgreen', 'lightcoral'])

plt.title('Sales Order Distribution by Channel')

plt.ylabel('')

plt.show()
